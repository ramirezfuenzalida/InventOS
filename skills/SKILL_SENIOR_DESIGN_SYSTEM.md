# 🎨 Skill: Diseño y Estética Visual Senior (Cosmos 2.0)

Este protocolo asegura que InventOS tenga una estética **Premium** de nivel profesional.

## Pilares del Diseño
- **Glassmorphism**: Uso de fondos translúcidos con desenfoque (`backdrop-filter: blur`) para dar profundidad en modales y paneles.
- **Motion Design**: Transiciones fluidas para entradas y salidas de componentes.
- **Paleta Armónica**: Uso de degradados sutiles (Grayscales premium + Accent Colors institucionales).
- **Tipografía**: Jerarquía clara con fuentes modernas y legibilidad optimizada para gestión de datos.

## Protocolo de Implementación
1. **Tokens de Diseño**: Uso estricto de variables CSS para consistencia de marca.
2. **Micro-interacciones**: Feedback visual inmediato en botones y tarjetas (hover, active).
3. **Responsive Excellence**: El diseño debe ser impecable tanto en dispositivos móviles (para escaneo QR) como en desktop (para administración).

## Regla de Oro
"La estética es una funcionalidad. Un dashboard ordenado y bello reduce la carga cognitiva del usuario."
